The system's primary container — hard 2px corners, hairline border, never a drop shadow.

```jsx
<Card label="You walk away with">
  <div className="dc-val">A shared written brief</div>
</Card>
<Card tone="ink">…</Card>
<Card tone="fill">…</Card>
```

Tones: `outline` (ink hairline on paper), `fill` (cream, recessed), `ink` (dark inverted), `plain` (faint rule). Pass `label` for the mono eyebrow, `pad` to override padding.
