Small pill-shaped chip with a mono uppercase label — version stamps, status flags, category tags.

```jsx
<Pill dot>05 Jun 2026 · V1.0</Pill>
<Pill tone="accent">In progress</Pill>
<Pill tone="soft">Strategy</Pill>
```

Tones: `outline` (ink border), `solid` (ink fill), `accent` (orange fill), `soft` (cream). Set `dot` for a leading status marker.
