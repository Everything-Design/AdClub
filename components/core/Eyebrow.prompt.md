Mono uppercase kicker that labels a section above its heading. The single most-used label element in the system.

```jsx
<Eyebrow>Engagement Plan / 2026</Eyebrow>
<Eyebrow tone="muted">Week one</Eyebrow>
```

Default tone is orange (`accent`). Use `muted` for page chrome, `ink` for emphasis on light grounds, `onInk` on dark surfaces.
