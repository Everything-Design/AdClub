Pill-shaped action button with a mono, uppercase, tracked label — use for the rare CTAs in an otherwise editorial system (download, sign-off, next step).

```jsx
<Button variant="primary" size="md" onClick={start}>Book the sprint</Button>
<Button variant="secondary">Read the approach</Button>
```

Variants: `primary` (filled orange), `secondary` (ink outline), `ghost` (text only), `inverse` (white on dark surfaces). Sizes: `sm` `md` `lg`. Pass `href` to render as a link. Keep labels short — two or three words.
