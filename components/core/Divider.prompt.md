The blueprint rule system. Dashed opens sections, dotted splits items inside a component, solid is a hard edge — mixing dashed and dotted on a page is intentional.

```jsx
<Divider variant="dashed" />
<Divider variant="dotted" tone="rule" />
<Divider variant="dashed" tone="onInk" />
```

Use `thick` for the 2px phase-grid top edge. `tone="onInk"` on dark surfaces.
